## ✅ 解决方案

### 方案1：添加缺失的 i18n 翻译
需要在 `i18n.js` 中为以下新增内容添加英文翻译：

```javascript
// 需要新增到 en 分支的翻译
hardwareCollect: {
  statusLabel: {
    idle: 'Not connected, please connect external device',
    connecting: 'Connecting...',
    connected: 'Connected · {name}'
  },
  disconnect: 'Disconnect',
  connect: 'Connect Device',
  stopCollect: 'Stop Capture',
  startCollect: 'Start Capture',
  connectHint: 'Please connect an external device first',
  collectingHint: 'Collecting data, please keep device stable...',
  realtimePhysiology: 'Real-time Physiological Data',
  // ...更多标签
}

aiChat: {
  brandName: 'AI Health Chat', // 已有
  clear: 'Clear', // 已有
  quickQuestions: [
    'I have been feeling palpitations lately, please analyze it',
    'How is my biological age assessment?',
    'How can I improve my cardiovascular health?',
    'What should I do about low oxygen saturation?',
    'Create an anti-aging diet plan for me'
  ]
}

// tabbar 翻译
tabbar: {
  home: 'Home',
  detection: 'Detection',
  report: 'Report',
  profile: 'Profile'
}

// 日期相关
dateFormat: {
  weekdays: ['Sun', 'Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat']
}
```

### 方案2：更新 Vue 文件
在各页面中使用 `t()` 函数替换硬编码文本：

**示例改进：** `hardware-collect/index.vue`
```vue
<!-- 改前 -->
<text class="nav-title">硬件采集</text>

<!-- 改后 -->
<text class="nav-title">{{ t('hardwareCollect.title') }}</text>
```
